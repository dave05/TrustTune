Why Calibration Matters

Uncalibrated models can lead to costly mistakes in high-stakes decisions.

Machine learning models often output scores that don't accurately reflect true probabilities. This miscalibration can lead to:

  • Overconfident predictions leading to risky decisions
  • Underconfident predictions causing missed opportunities
  • Inconsistent risk assessments across different models
  • Poor performance in downstream decision-making systems

TrustTune solves these problems by transforming your model's outputs into reliable probabilities you can trust.

Documentation

Quick Start with Python
from trusttune.core.platt import PlattCalibrator

# Initialize calibrator
calibrator = PlattCalibrator()

# Fit calibrator with your data
calibrator.fit(scores, labels)

# Get calibrated probabilities
calibrated_scores = calibrator.predict_proba(scores)

Install the package using pip:

pip install trusttune
API Usage
import requests
import json

# Prepare data
data = {
    "scores": [0.2, 0.7, 0.9],
    "labels": [0, 1, 1],
    "calibrator_type": "platt"
}

# Call API
response = requests.post(
    "http://localhost:8000/calibrate",
    json=data
)

# Get results
result = response.json()
print(result["calibrated_scores"])
Docker Deployment
# Build the Docker image
docker build -t trusttune .

# Run the container
docker run -p 8000:8000 trusttune
